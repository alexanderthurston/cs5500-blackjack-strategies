# cs5500-blackjack-strategies

To build and run the code, the following lines are used:
• mpic++ blackJack.cpp
• mpiexec -n 4 a.out

note: more than one node is required, if only one node is used an error message will appear

Expected output:
The program will print out all the results to the terminal on completion
The program will print the results to SimOut.txt as well, which can be easily imported to excel

There are no additional inputs to the program
